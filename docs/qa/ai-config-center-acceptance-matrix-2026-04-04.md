## 7. API 接口对应
| 操作 | 接口 |
|------|------|
| 获取供应商列表 | `GET /v1/admin/ai/providers` |
| 创建/编辑/删除供应商 | `POST/PUT/DELETE /v1/admin/ai/providers/:id` |
| 获取/添加/删除模型 | `GET/POST/DELETE /v1/admin/ai/providers/:id/models` |
| 从远程同步模型 | `POST /v1/admin/ai/providers/:id/models/sync` |
| 获取/添加/删除凭证 | `GET/POST/DELETE /v1/admin/ai/providers/:id/credentials` |
| 连接测试 | `POST /v1/admin/ai/providers/:id/test` |
